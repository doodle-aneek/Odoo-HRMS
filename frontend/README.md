# HRMS Frontend

This React + Vite frontend provides a role-based UI for the Odoo HRMS backend.

## Structure
- src/pages/auth: sign-in and sign-up views
- src/pages/dashboard: employee and admin dashboards
- src/pages/profile: profile management view
- src/pages/attendance: attendance check-in and history
- src/pages/leaves: leave requests and workflow
- src/pages/payroll: payroll and payslip views
- src/services: mock API service layer for auth and leaves

## Run locally
1. Install Node.js and npm.
2. Run `npm install` in the frontend folder.
3. Run `npm run dev`.
